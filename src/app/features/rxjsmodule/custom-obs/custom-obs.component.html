Custom Observable

  • Observable : An observable is a producer of values or events over time. It represents a data stream that can emit zero or more values, complete, or produce an error. Observables are lazy, meaning they don't start emitting values until someone subscribes to them.
  • Observer : An observer is a consumer of values delivered by an observable. It is an object that defines how to handle the values emitted by the observable, as well as how to handle completion and errors.
